# SpringBootRest
## This is a **Gradle** project for **Spring Boot** based web service ##
  
## **Build** ##
`gradlew build`

## **Run** ##
`gradlew bootRun`

## **Generate Eclipse Project** ##
`gradlew eclipse`

## **Generate Idea Project** ##
`gradlew idea`

## Libraries++ ##
*`springfox.documentation : Automated JSON API documentation for API's built with Spring.` 

*`junit : Unit tests`
